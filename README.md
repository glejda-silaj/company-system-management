# NovaNet Web Application

NovaNet is a web application designed for company data management. It provides an admin dashboard for managing and organizing business data in an efficient and user-friendly way.

## 🚀 Features

- Admin dashboard interface
- Create, Read, Update, Delete (CRUD) operations
- User management system
- Database integration
- Responsive web design

## 🛠️ Technologies Used

- HTML, CSS, JavaScript
- PHP (or backend tech you used)
- MySQL / Database system
- Bootstrap (if used)

## 📂 Project Structure

- /admin - Admin dashboard files  
- /assets - CSS, JS, images  
- /includes - Backend logic and connections  
- index.php - Main entry point
